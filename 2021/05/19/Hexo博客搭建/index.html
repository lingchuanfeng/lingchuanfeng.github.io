Hexo博客搭建

软件环境

Node.JS

node.js官方网站：http://nodejs.cn/download/

node.js阿里云镜像站：https://npm.taobao.org/mirrors/node/

Git

Git官网(下载速度可能稍慢)：https://git-scm.com/

Git阿里云镜像站（推荐）：https://npm.taobao.org/mirrors/git-for-windows/

Hexo

官方网站：https://hexo.io/zh-cn/

Github

官方网站：https://github.com/

检验

node.js 的检查

node -v

npm的检查

npm -v

Git 的检查

git --version

以上有显示版本号，则说明安装成功

HEXO的安装

假设下边操作是在D；盘完成

  1. 在D盘启动cmd
  2. 安装之前可以先设置一下淘宝镜像加速器

    npm install -g cnpm --registry=https://registry.npm.taobao.org

  3. 全局安装框架

    npm install hexo-cli -g

  4. 创建你的博客目录

    hexo init 你博客的文件夹名字

  5. 进入你博客的目录

    cd 你博客的文件夹名字

  6. 复制文件到你博客的目录

    npm install

  7. 安装Hexo部署插件（博客目录下启动）

    npm install hexo-deployer-git --save
